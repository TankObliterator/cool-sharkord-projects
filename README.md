# Cool Sharkord Projects
A hand-curated list of open, free, and useful [Sharkord](https://sharkord.com/) projects — perfect for enhancing the user/admin experience.

>🛑 DISCLAIMER: While I personally vet these projects, I am not a developer and cannot guarantee the security of the underlying code.
>Please review projects yourself before use and [report](https://github.com/TankObliterator/cool-sharkord-projects/issues/new/choose) any issues you find.
